News

  • March 29, 2010: gtkmm 2.20.0 and glibmm 2.24.0 released, as per the GNOME schedule. gtkmm 2.20.0 wraps new API in GTK+ 2.20 and is API/ABI-compatible with gtkmm >= 2.4. It is a version of the gtkmm-2.4 API with API additions.
  • September 22, 2008: gtkmm 2.14.0 released, as per the GNOME schedule. gtkmm 2.14.0 wraps new API in GTK+ 2.14 and is API/ABI-compatible with gtkmm 2.12, 2.10, 2.8, 2.6 and 2.4. It is a version of the gtkmm-2.4 API, with API additions.
  • September 14, 2007: gtkmm 2.12.0 and glibmm 2.14.0 released, as per the GNOME schedule. gtkmm 2.12.0 wraps new API in GTK+ 2.12 and is API/ABI-compatible with gtkmm 2.10, 2.8, 2.6 and 2.4. It is a version of the gtkmm-2.4 API, with API additions.
  • August 2006: gtkmm and glibmm 2.10.0 released, as per the GNOME schedule. gtkmm 2.10.0 wraps new API in GTK+ 2.10 and is API/ABI-compatible with gtkmm 2.8, with API additions.
  • September 2005: gtkmm and glibmm 2.8.0 released, as per the GNOME schedule. gtkmm 2.8.0 wraps new API in GTK+ 2.8 and is API/ABI-compatible with gtkmm 2.6, with API additions.
  • March 2005: gtkmm and glibmm 2.6.0 released, as per the GNOME schedule. gtkmm 2.6.0 wraps new API in GTK+ 2.6 and is API/ABI-compatible with gtkmm 2.4, with API additions.
  • April 2004: gtkmm and glibmm 2.4.0 released, as per the GNOME schedule. gtkmm 2.4.0 wraps new API in GTK+ 2.4 and installs in parallel with gtkmm 2.2.
  • October 2003: gtkmm 2.3.0 was released. This will become the new stable, parallel-installable, gtkmm 2.4 in March 2004.
  • March 2003: libsigc++ article in Dr Dobb's Journal.
  • January 2003: gtkmm 2.2.0 released, API/ABI-compatible with gtkmm 2.0, with API additions.
  • December 2002: libglademm 2.0.0, gconfmm 2.0.0 and libgnomecanvasmm 2.0.0 released - these APIs are binary-frozen.
  • October 2002: gtkmm 2.0.0 is released - the API is binary-frozen.
  • September 2002: gtkmm article in Software 2.0 (Polish) magazine.
  • May 2002: gtkmm maintainer interviewed by Linux Orbit.
  • April 2002: gtkmm 1.2 article in C/C++ User's Journal.
  • April 2002: gtkmm2 presentation at GUADEC3 in Seville.
  • October 2001: Al Stevens reviews gtkmm 1.2 favourably in Dr Dobb's Journal.
