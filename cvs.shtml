CVS

CVS is a powerful tool for maintain version control with multiple developers produced by Cyclic. Both the stable and developer versions of gtkmm are housed in the Gnome CVS repository.

Documentation for CVS is available at Introduction to CVS or the CVS manual. CVS is available on a wide number of platforms including Unix, Window 95/NT, and Macintosh.

Anonymous CVS

The main CVS repository is located at cvs.gnome.org and requires a developer login. For those who do not have a login on the gnome developer tree, anonymous CVS checkouts are available from anoncvs.gnome.org.

To check out the latest development version: 

cvs -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome login 
(hit return as the password)
cvs -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome co gtkmm-1.3

The stable version (with any bug fixes made between releases) is also available. 

cvs -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome login 
(hit return as the password)
cvs -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome co gtk--

You will need to run the autogen.sh script to generate the build files.
