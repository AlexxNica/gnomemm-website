CVS

CVS is a powerful tool for maintain version control with multiple developers produced by Cyclic. Both the stable and developer versions of gtkmm are housed in the Gnome CVS repository. Additional information about cvs.gnome.org is available at developer.gnome.org.

Anonymous CVS

The main CVS repository is located at cvs.gnome.org and requires a developer login. For those who do not have a login on the gnome developer tree, anonymous CVS checkouts are available from anoncvs.gnome.org.

To check out the latest version of gtkmm: 

cvs -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome login 
(hit return as the password)
cvs -z3 -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome co gtkmm2

The older 1.2 version of gtkmm (with any bug fixes made between releases) is also available. 

cvs -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome login 
(hit return as the password)
cvs -z3 -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome co gtk--

To check out the latest development version of gnomemm: 

Login as with gtkmm and:
cvs -z3 -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome co gnomemm

The stable 1.2 version of gnomemm (with any bug fixes made between releases) is also available. 

Login and:
cvs -z3 -d :pserver:anonymous@anoncvs.gnome.org:/cvs/gnome co gnome--

You will need to run the autogen.sh script to generate the build files.
