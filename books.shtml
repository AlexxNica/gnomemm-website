Recommended Books

The gtkmm developers find these books helpful:

Core books

  • The C++ Programming Language, Bjarne Stroustrup
  • The C++ Standard Library, Nicolai M. Josuttis
  • Design Patterns, Erich Gamma et al

For beginners

  • Accelerated C++ - Practical Programming by Example, Andrew Koenig & Barbara E. Moo

Others

  • Advanced CORBA Programming with C++, Michi Henning & Steve Vinoski
  • GNU Autoconf, Automake, and Libtool, Gary Vaughan et al
  • Programming With Gnu Software, Michael Kosta Loukides et al
