Available widgets

In addition to the wrappers distributed with gtkmm, additional widgets are available from other sources.

  • See GTK+ widget repository
  • GtkGLArea-- an OpenGL widget that works with gtkmm
  • Panel--, GNOME panel applet wrapper by Gerg? ?rdi
  • GtkExtra--, Wrappers for GtkExtra (sheet, etc)
  • GConf--, Wrappers for GConf, particularly GConfClient

gtkmm tools

  • Glade-- - Module for Glade the GTK+ GUI builder
  • Gladecc Another module for Glade
  • Bakery- Document/View Model with XML
