Additional Resources

Available widgets

In addition to the wrappers distributed with gtkmm and gnomemm, additional widgets are available from other sources.

  • GtkGlExtmm OpenGL rendering for gtkmm
  • gtkmozembedmm Mozilla-based web browser widget
  • Plotmm Plot widget for scientific applications
  • gtksourceviewmm, Wrappers for GtkSourceView (Gtk::TextView with syntax highlighting)
  • VTEmm Terminal widget
  • gtkextramm, Wrappers for GtkExtra (plot, sheet, etc)
  • FlowCanvas Canvas for data flow systems.

gtkmm tools

  • Glade - User Interface Designer for GTK+. Glade UI files can be used from a gtkmm program by using libglademm.
  • glademm - Glade Module for automatically generating gtkmm code. (Note: glademm is different than libglademm)
  • Bakery - Document/View Model
