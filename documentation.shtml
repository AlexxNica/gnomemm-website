Documentation Overview

This documentation is for the gtkmm 2.4 and gnomemm 2.6 APIs. There is also documentation for the gtkmm 2.0/2.2 APIs.

Note that the version numbers here refer to API / ABI compatibility versions and not release versions. gtkmm has maintained API and ABI compatibility since the 2.4 release, so all subsequent releases are backwards compatible and are known generically as "gtkmm-2.4".

Getting Started with gtkmm

If you've never used gtkmm before, you may want to start by reading the tutorial-style online book Programming with gtkmm. This book is available for free online and was written by gtkmm authors and users. The book is also included as part of the gtkmm download for those that want to have a local copy to reference. If you have suggestions for improving the book, feel free to contact us.
