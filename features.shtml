Supported Features

Widget wrapping framework

  • Automatic wrapping system for converting foreign resources
  • Access to GTK+ resources for integrating GTK+ tools
  • Consistent use of reference semantics for parameter passing
  • OO technologies for combining user interface elements
    • Inheritance
    • Polymorphic references
    • Type-safe callbacks
  • Complete memory management
    • Object composition
    • Automatic deallocation of dynamically managed memory

gtkmm widgets

  • Tracks GTK+ development
  • Single inheritance widget hierarchy
  • User defined C++ widgets support
  • Hundreds of C++ wrapper classes
  • Standard string class support
  • STL iterator and list support
  • Factories for constructing Toolbars and Menus

gnomemm widgets

  • Full Canvas support
  • Large number of predefined menu items and icons
  • Preconfigured dialogs
  • MDI support

Type-safe signal system

(via sigc++)
  • One-to-Many callbacks
  • Callbacks to:
    • Member functions
    • Static member functions
    • Function objects
    • Signals
  • Invalidation of connections when target is deleted
  • Passing extra data from connect to the callback
  • Proxies for GTK+ connectivity
