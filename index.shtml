New: gtkmm 2

gtkmm2 is not yet as complete as gtkmm 1.2, but it's already easier to use. And we also have a GNOME2 version of gnomemm.
Here's an overview, with documentation and status.

gtkmm

gtkmm (previously known as Gtk--) is the official C++ interface for the popular GUI library GTK+. Highlights include typesafe callbacks, widgets extensible via inheritance and a comprehensive set of widget classes that can be freely combined to quickly create complex user interfaces. gtkmm is free software distributed under the GNU Library General Public License (LGPL).

gnomemm

gnomemm (Previously known as Gnome--) is a set of powerful C++ bindings for the GNOME libraries, which provide additional functionality above GTK+/gtkmm.

libsigc++

Also, be sure to check out our companion library, libsigc++. Due to popular demand our signal/slot library is now provided separately for use in non-GUI code. Thus all the power of typesafe callbacks and flexible adaptor classes is available for use, without changes to the C++ language.

Other gtkmm-related libraries

Be sure to check out Bakery a framework for constructing gnomemm applications. It allows use of the Document/View architecture. Also for gtkmm, GtkExtra-- for those extra widget wrappers.
