gtkmm

gtkmm is the official C++ interface for the popular GUI library GTK+. It closely follows GTK+ development. Highlights include typesafe callbacks, widgets extensible via inheritance and a comprehensive set of widgets. You can create user interfaces either in code or with the Glade designer, using libglademm. Here's an overview, with documentation and status. gtkmm is free software distributed under the GNU Library General Public License (LGPL).

The new gtkmm 2.4 has recently become API/ABI stable, and can be installed in parallel with the previous stable gtkmm 2.2. gtkmm development is synchronised with GTK+. We follow the official GNOME Platform Bindings release schedule.

gnomemm

There is also a set of powerful C++ bindings for the GNOME libraries, which provide additional functionality above GTK+/gtkmm.

libsigc++

Also, be sure to check out our companion libraries, glibmm (UTF8 strings, threading, etc) and libsigc++ (signal/slots).

Other gtkmm-related libraries

Be sure to check out Bakery a framework for constructing gtkmm/gnomemm applications. It allows use of the Document/View architecture.
