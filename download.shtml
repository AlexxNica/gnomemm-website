Downloading

Source

The source packages for gtkmm are available on the sourceforge download area.

Requirements

  • GTK+
  • libsigc++ (1.2.x for gtkmm 2, or 1.0.x for gtkmm 1.2)

Here are dependency graphs for gtkmm and gnomemm, generated automatically with jhbuild and graphviz.

GTK+

You can download GTK+ and GNOME source code from download.gnome.org.

Binary

You might find a variety of 3rd party RPMs and debs on the internet, but only the ones listed here are approved by the gtkmm team. Please encourage your Linux distribution to provide official gtkmm packages.

gtkmm 2.2:

  • Debian Linux: apt-get install libgtkmm2.0-dev. These project pages might give status.
  • Fedora/RedHat Linux: Use yum: yum install gtkmm2-docs to get the runtime, development, and documentation packages.
  • Mandrake Linux: They will be in the next release of the distro. They are already in the beta.
  • Windows: Here is an installer.
  • MacOS X: There is a Fink package.

gtkmm 1.2

  • RedHat 7.3: Ximian
  • Debian: apt-get install libgtkmm-1.2-dev
